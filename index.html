Summary: Latitude VS. X

The purpose of this project was to analyze how weather changes as approaching the equator. To accomplish this analysis, the data was retrieved from the OpenWeather API to assemble a dataset on over 500 cities.

After assembling the dataset, applying Matplotlib to plot various aspects between the weather factors and latitude. Factors included: temperature (F), cloudiness (%), wind speed (mph), and humidity (%). This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
